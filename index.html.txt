        <section id="tipos_de_uñas">
            <h2>Tipos de Uñas</h2>
            <p>Descubre los diferentes tipos de uñas y cómo se hacen:</p>
            <ul>
                <li>Uñas de Soft Gel: Esta técnica implica aplicar una capa de gel flexible sobre la uña natural o extensión para crear una base duradera y resistente. El gel se cura bajo una lámpara LED o UV para endurecerlo y luego se moldea y se da forma según el estilo deseado.</li>
                <li>Uñas Semi Permanente: Consiste en aplicar esmaltes de gel de alta calidad que se curan bajo una lámpara LED o UV para crear un acabado duradero y brillante. Después de cada capa de esmalte, se cura bajo la lámpara para garantizar un acabado impecable y duradero.</li>
            </ul>
